# X2-StarlightXpressAO

TheSkyX X2 plugin for the Starlight Xpress AO tip-tilt mirror unit.

Builds are produced automatically by GitHub Actions for Linux, macOS, and Windows. See the [releases page](../../releases) for pre-built binaries.

- [AGENTS.md](AGENTS.md) — project guide for contributors and AI agents (purpose, layout, code style, build instructions)
- [NOTES.md](NOTES.md) — binary analysis methodology, reconstructed undocumented interfaces, and serial protocol reference
